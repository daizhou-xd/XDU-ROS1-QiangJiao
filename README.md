# 西安电子科技大学个人学习记录
机器人工程专业

机器人操作系统与应用|Code&Notes

## 课程简介
本课程主要介绍机器人操作系统（ROS）的基本概念、架构和应用。

## 课程框架
* 第一章 ROS概述与环境搭建
* 第二章 ROS通信机制
* 第三章 ROS架构与运行管理
* 第四章 ROS常用组件
* 第五章 机器人建模与仿真
* 第六章 ROS进阶功能

## 环境搭建
1. 安装VMware虚拟机
2. 安装Ubuntu 18.04 桌面版
3. 安装ROS Melodic
4. 安装Vscode编辑器等工具
